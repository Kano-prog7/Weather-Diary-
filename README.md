# Weather Diary

**Автор:** Кондратьев Сергей

## Описание
Приложение "Дневник погоды" позволяет вести учет погодных условий: добавлять записи о температуре, осадках и описании, а также фильтровать их по дате и температуре. Данные сохраняются в формате JSON.

## Запуск приложения
1. Установите Python 3.x.
2. Установите необходимые библиотеки:
   `pip install tkcalendar`
3. Запустите скрипт:
   `python main.py`

## Примеры использования
1.  Введите дату, температуру, описание и отметьте осадки.
2.  Нажмите "Добавить запись".
3.  Используйте фильтры "По дате" или "По температуре" для поиска нужных данных.
